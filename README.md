listoffunctions
===============
関数をQuickfixリストを使用して一覧表示したり一覧から選択して移動できる。

インストール
===============
NeoBundle 'mareku/listoffunctions' 

使い方
===============
    :listoffunctions
と入力すればQuickfixリストに関数一覧が表示される。  
リスト内で j/k や矢印で関数を選択して、 
Enter すれば移動する。

対応言語
===============
    php
    python
    vim
    vb
